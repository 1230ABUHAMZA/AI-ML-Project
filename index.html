📏 Object Size Calculator

Upload an image and measure object dimensions with precision

Enter a known dimension (e.g., A4 paper width = 21 cm)

📌 How to Use:

  1. Upload an image containing the object you want to measure
  2. Enter a known reference dimension (e.g., the width of a book)
  3. Click on 4 corner points anywhere around the object in ANY ORDER
  4. Points will automatically arrange to form a proper quadrilateral
  5. Click "Calculate Size" to get the dimensions
